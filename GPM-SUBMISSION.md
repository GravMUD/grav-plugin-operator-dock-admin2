## Summary

Operator Dock for Admin2 is a free operator utilities plugin for Grav 2's Admin2 panel. It merges configurable header shortcuts into `menubarLinks`, registers an Operator Launch Pad dashboard widget (Pages, Plugins, Themes, Tools, Settings + custom links), and adds a one-click Clear Cache menubar action. Optional Team DC shortcut pack (GetGRAV!, Mud Bazaar). Pairs with JavaBean for Admin2 (theming).

## Dependencies

- grav >= 2.0.0
- admin2 >= 1.0.0
- api >= 1.0.0

bin/gpm install grav-operator-dock-admin2

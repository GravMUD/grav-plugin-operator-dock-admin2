Header shortcuts

Merges into Admin2 menubarLinks — View Site, Grav Learn, custom URLs, optional Team DC pack.

🛸

Launch Pad widget

Dashboard tiles for Pages, Plugins, Themes, Tools, Settings — plus your configured shortcuts and live site vitals.

🧹

Clear Cache

One-click menubar action. Requires api.system.write. No terminal cosplay required.

⚙️

Settings panel

Admin2 → Settings → Operator Dock, or the sidebar plugin page. Toggle every feature independently.

🌓

Andy-compatible

Official Admin2 extension points only — same patterns as JavaBean. Uninstall cleanly, no ghost routes.

☕

Pairs with JavaBean

Paint with JavaBean. Operate with Operator Dock. Commercial GravMUD plugins stay separate products.

Install

Two commands to dock.

$ bin/gpm install operator-dock-admin2 # Or manual: extract to user/plugins/operator-dock-admin2, clear cache, enable in Admin2 → Plugins.
